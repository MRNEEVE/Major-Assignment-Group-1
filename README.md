INVENTORY TRACKER SYSTEM
Project Overview
A web-based inventory management system that helps businesses track products, manage stock levels, and generate reports. The system allows users to add, edit, delete, and search products while maintaining data persistence using browser localStorage.

 Features
- Add new products with name, category, quantity, and price
- View all products in a table format
- Edit existing product details
- Delete products from inventory
- Search products by name or category
- Dashboard with statistics
- Export inventory to CSV
- Responsive design for mobile devices
- Data persistence using localStorage

Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API
- Git & GitHub for version control

How to Run
1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. No installation or dependencies required
